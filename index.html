Bookmark install page

Install your student dashboard bookmark in a minute.

This page helps you install a bookmark, not a browser extension. Once you save it, click it on Canvas to open a smarter dashboard for your real classes, missing work, due dates, tests, and study prompts.

Drag This Bookmark

Press copy, or drag the bookmark button to your bookmarks bar.

Fastest install

  1. Show your bookmarks bar in Chrome with Cmd + Shift + B if it is hidden.
  2. Drag Drag This Bookmark to the bookmarks bar.
  3. Open your Canvas website and sign in.
  4. Click the new bookmark to open your dashboard.

Manual install

  1. Create a new bookmark in your browser.
  2. Name it Student Dashboard.
  3. Click Copy bookmark code below.
  4. Paste that code into the bookmark URL field.
  5. Save it, then click it while you are on Canvas.

Chrome

Open the page, show your bookmarks bar, then drag the bookmark button onto it.

Safari

Show the Favorites Bar, make a new bookmark, then paste the copied code into the address field.

What it filters

It tries to keep real classes and hide obvious filler spaces like advisory, sandbox, or orientation courses.

What happens when you click it

The bookmark opens a fullscreen Canvas dashboard with your classes, a needs-attention section, missing assignments, work due this week, upcoming tests and quizzes, local done-tracking, and one-click study prompts.

If your school locks down some Canvas data, a section may show an error message instead of data. Study prompts are designed to help you learn the material, not submit work dishonestly.
